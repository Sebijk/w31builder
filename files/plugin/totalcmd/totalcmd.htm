Mini-Windows 3.1 Builder plugin

Total Commander v6

Total Commander (by Christian Ghisler), ist ein Shareware Datei-Manager für Windows 3.1/95/98/ME/NT/2000/XP.
Bitte holt euch für Windows 3.1 Light Builder die 16-Bit Version herunter (Windows 3.x)

Wenn du eine Registrierte Version von Total Commander benutzt dann kopiere die Datei wincmd.key in dein plugin\totalcmd Verzeichnis.
Windows 3.1 Light Builder kopiert diese Datei automatisch, wenn diese Datei existiert.
