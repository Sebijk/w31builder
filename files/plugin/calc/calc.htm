Mini-Windows 3.1 Builder plugin

Rechner

Holen Sie sich als erstes den Windows 3.1 Rechner herunter.
Dieses Plugin fügt den Rechner hinzu. Da der Rechner Bestandteil von Windows 3.1x ist müssen Sie die Datei "calc.exe" von Windows
3.1/3.11 holen. Es ist aber ein Update für den Windows 3.1x erschienen. Denn die Vorgängerversionen des Taschenrechners zeigten
einfache Subtraktionsfehler, wenn man zwei Zahlen von einander subtrahierte, welche Dezimalstellen enthielten. Bei der Anzeige der Lösung wurde nicht korrekt gerundet.
Laden Sie das Update unter ftp://ftp.microsoft.com/Softlib/MSLFILES/WW1138.EXE herunter, und entpacken Sie die Datei und fügen "calc.exe" in dieses Verzeichnis ein.
Fügen Sie ggf. die Datei "calc.hlp" unter Windows-Hilfe Plugin ein (nicht im Update enthalten).
